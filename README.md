#tiger 
Instruction for spin VM and destroy: 
1. Install vagrant 
2. Install virtualbox 
3. Run 'vagrant up' 
4. Run 'vagrant ssh'
5. Open the browser and type "localhost:5000" in the URL
6. At the end of work, logout from the machine (type exit) 
7. Run 'vagrant destroy -f'<br/>
