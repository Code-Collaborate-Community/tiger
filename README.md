# tiger 
Instruction for spin VM and destroy: 
1. Install vagrant 
2. Install virtualbox 
3. Run 'vagrant up' 
4. Run 'vagrant ssh' 
5. In the end of work, logout from the machine 
6. Run 'vagrant destroy -f'
